영업 미팅에서 본 데모는 정말 대단했죠. 하지만 실제로 우리 데이터를 넣으니까 이상한 답변, 환각, 느린 속도... 완전히 다른 결과물.

"3개월 개발 끝에 나온 결과가 GPT API 그대로 쓰는 것보다 못했어요." — 중견 제조사 DX팀장

"비용이 예상의 5배..."

"원가 계산을 다시 해보니 건당 처리 비용이 수익을 넘어가더라고요." — 핀테크 스타트업 CTO

"인수인계 없이 외주 개발자가 퇴사하니 유지보수 자체가 불가능." — 헬스케어 스타트업 대표

"전문가라더니..."

'AI 전문가'라고 해서 맡겼는데, 알고 보니 ChatGPT API 래핑만 할 줄 아는 수준. 프롬프트 엔지니어링? RAG? 전부 외주.

정말 우리 업무에도 AI가 적용 가능할까요?
솔직히, 모든 업무에 AI가 적합한 건 아닙니다. 그래서 1주일의 PoC 기간을 반드시 거칩니다. 실제 데이터로 테스트해보고 "이건 AI로 해결하기 어렵습니다"라고 정직하게 말씀드리기도 합니다.
API 비용이 너무 많이 나오면 어떡하죠?
저희 강점 중 하나가 비용 최적화입니다. 프롬프트 압축, 캐싱 전략, 로컬 모델 활용 등으로 토큰 사용량을 평균 60% 이상 줄입니다. 계약 전 비용 시뮬레이션을 제공합니다.
개발 후 유지보수는 어떻게 되나요?
프로덕션 패키지에는 12개월 유지보수가 기본 포함입니다. 모델 성능 모니터링, 드리프트 감지, 이슈 대응까지 책임집니다. 12개월 후엔 완전한 기술 이전과 교육을 진행합니다.
다른 업체에서 실패한 프로젝트도 맡나요?
네, 오히려 그런 프로젝트를 많이 맡았습니다. "AI 프로젝트 구조대"라는 별명도 있어요. 기존 코드와 데이터를 분석하고, 왜 실패했는지 진단한 뒤 현실적인 해결책을 제시합니다.
