DEVELOPERS

개발자를 위한 빠른 시작

5분 안에 첫 번째 API 호출을 완료하세요

⚡ Quickstart (5분)

Python 
import haiv

client = haiv.Client(api_key="YOUR_API_KEY")

# 음성 파일 업로드 & 전사
result = client.transcribe(
    file="meeting.mp3",
    speaker_diarization=True,
    language="ko"
)

# 결과 출력
for segment in result.segments:
    print(f"[{segment.speaker}] {segment.text}")

📡 API 기능

POST /v1/transcribe 음성 → 텍스트
POST /v1/diarize 화자분리
POST /v1/summarize 요약
WS /v1/stream 실시간 스트리밍
