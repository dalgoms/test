# 메가메뉴 구조 개선 리포트

**작성일**: 2026년 2월 2일  
**작성자**: 이세영 팀장  
**작업 범위**: 메가메뉴 전체 구조 정리 및 문구 개선

---

## 1. 메가메뉴 구조 개선 — Before / After

| 구분 | Before | After |
|------|--------|-------|
| AI 미디어 상위 | AI 미디어 | AI 미디어 클립데스크 |
| 제품 인식 | 개별 기능 나열 | AI 미디어 제품군(플랫폼) 인식 |
| 하위 메뉴 방식 | Caption / Translate / Edit (내부 제품명 중심) | 실시간 AI 자막 / 영상 자막 번역 / AI 기반 영상 편집 (기능 중심) |
| 사용자 이해도 | 기능 의미 추측 필요 | 첫 화면에서 바로 이해 가능 |
| 전체 톤 정합성 | 일부 B2C 서비스 느낌 | 전체 B2B 솔루션 톤 통일 |
| Clipdesk 포지션 | 단일 서비스 | AI 미디어 대표 제품군 |
| 확장성 | 기능 추가 시 구조 애매 | 향후 기능 확장 자연스럽게 수용 가능 |

### 핵심 메시지

> Before는 내부 명칭 중심 구조였다면,  
> After는 고객 관점에서 **"무엇을 할 수 있는지"**가 보이도록 재설계했습니다.  
> Clipdesk를 서비스가 아닌 **AI 미디어 제품군**으로 포지셔닝한 구조입니다.

## 2. 상세 변경 내역

### 2.1 삭제된 메뉴

| 삭제 항목 | 비고 |
|-----------|------|
| LLM · RAG 기반 지식 AI 플랫폼 | 인공지능 섹션에서 삭제 |
| AI 기록·자막 전체 (SORIBARO, TEXTAR, SORIZAVA, WEBFOS) | 컬럼 전체 삭제 |
| WORKSFY | AI 미디어에서 삭제 |
| SORIZAVA ACADEMY | AI 미디어에서 삭제 |

### 2.2 인공지능 섹션

| 메뉴 | 서브 문구 |
|------|----------|
| 음성인식 · 음성합성 AI 솔루션 | HAIV – 외부 AI 연동 음성 인식·합성 엔진 |
| 대규모 AI 모델 개발 | RAG 기반 지식 구축 · 업무 시나리오 설계 |
| AI 메타휴먼 수어 아바타 | AI 수어 서비스 리나 & 바로 |

### 2.3 AI 미디어 클립데스크 섹션

| 메뉴 | 서브 문구 |
|------|----------|
| 실시간 AI 자막 · 기록 자동화 | STT 기반 실시간 자막 생성 및 기록 전문가 검수 |
| 영상 자막 다국어 번역 | AI 번역 + 현지 원어민 번역 워크플로우 |
| AI 기반 영상 편집 | 텍스트 기반 편집 + 영상 전문가 제작 지원 |

### 2.4 홍보센터 섹션 (구 자료 센터)

| 변경 전 | 변경 후 |
|---------|---------|
| 자료 센터 | 홍보센터 |
| 콘텐츠 허브 | 팀벨 소식 |

| 메뉴 | 서브 문구 |
|------|----------|
| 고객 사례 | 공공·금융·기업 도입 성공 사례 |
| 최신 소식 | 보도자료 · 이벤트 · 기술 블로그 |

### 2.5 회사소개 섹션

| 메뉴 | 서브 문구 |
|------|----------|
| 사업 영역 | 인공지능 · AICC · AI 회의록 · AI 미디어 |

## 5. 기대 효과

1. **사용자 이해도 향상**: 기능 중심 메뉴명으로 첫 화면에서 바로 이해 가능
2. **B2B 톤 통일**: 전체 메뉴가 기업 솔루션 느낌으로 일관성 확보
3. **확장성 확보**: 향후 Clipdesk 신규 기능 추가 시 자연스러운 메뉴 확장 가능
4. **브랜드 인식 강화**: Clipdesk를 AI 미디어 대표 제품군으로 포지셔닝
