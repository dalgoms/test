팀벨이
기업의 기록 업무를 책임집니다.

바로노트는 마이크 또는 녹음파일을 통해 수집된 대화 내용을 자체 개발 AI 기술로 자동 기록 및 요약해 주는 솔루션입니다.

회의록, 상담일지, 조서, 보고서 등 다양한 기록 업무 시간을 단축하여 더 중요한 업무에 집중할 수 있도록 도와드립니다.

[제품 소개 이미지]
Feature 01

AI 요약 및 키워드 추출

AI 요약 기능이 대화 전체를 요약하고 다빈도 키워드를 자동으로 추출합니다. 요약된 내용은 복사하기 또는 파일 내보내기로 회의록·보고서·조서 작성 시 활용이 가능합니다.

  • 긴 회의 내용을 핵심 포인트로 자동 요약
  • 다빈도 키워드 자동 추출 및 태그 생성
  • 다양한 포맷으로 내보내기 지원
[AI 요약 기능 스크린샷]
Feature 02

실시간 음성 기록 및 화자 분리

대화 내용을 실시간 텍스트로 확인하고, 참석자별로 발언을 자동 구분합니다. 누가 무슨 말을 했는지 명확하게 기록하여 회의록 작성이 더욱 편리해집니다.

  • 실시간 음성-텍스트 변환 (STT)
  • AI 기반 화자 자동 분리
  • 타임스탬프 자동 생성
[화자 분리 기능 스크린샷]
Feature 03

주요 내용 북마크 및 메모

북마크로 중요 단어 및 문장을 표시해 빠르게 기록 확인이 가능합니다. 대화 내용을 실시간으로 확인하면서 편집 또는 메모를 추가할 수 있습니다.

  • 중요 발언 북마크 기능
  • 실시간 메모 및 편집
  • 북마크 기반 빠른 검색
[북마크 및 메모 기능 스크린샷]
Feature 04

파일 · 폴더 한번에 업로드

여러 개의 녹음 파일 또는 폴더를 한번에 업로드하여 기록으로 변환할 수 있습니다. 바로노트가 파일을 변환하는 동안 자유롭게 다른 업무를 진행하세요.

  • 다양한 오디오 포맷 지원
  • 대용량 파일 일괄 업로드
  • 백그라운드 변환 처리
[파일 업로드 기능 스크린샷]
