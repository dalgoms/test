2. 월별 부재율 발생 원인: 성과 문제가 아닌 유입 구조 문제

2.1 논리적 증명 프레임워크

[기존 해석 - 오류]
높은 부재율 → 상담팀 역량 부족 → 인력 충원/교육 필요

[구조적 해석 - 정확]
높은 부재율 → 근무외 시간 유입 과다 → 물리적 응대 불가 → 구조적 한계

2.2 증명 논거

논거 1: 물리적 응대 가능 시간의 제약

항목 수치
하루 총 시간 24시간
근무시간 9시간 (09:00~18:00)
실질 즉시응대 가능시간 5~6시간
응대 가능 비율 24시간 중 20.8~25%
상담팀이 100% 완벽히 응대해도, 하루의 최대 25%만 커버 가능

논거 2: 비대면 유입의 시간대 불일치

  • 비대면 상담 유입의 53.7%가 근무시간 외
  • 심야(0~6시) 유입만 4,283건(27.4%)
  • 이 시간대는 물리적으로 응대 인력 배치 불가

논거 3: 부재율 공식의 구조적 함정

기존 부재율 = 미응대 건수 / 전체 유입 건수

→ 분모에 '근무시간 외 유입'이 포함됨
→ 애초에 응대할 수 없는 건수가 포함된 계산
→ 부재율은 구조적으로 50% 이상 나올 수밖에 없음

구분 기존 부재율 문제점
공식 미응대 / 전체유입 분모에 응대 불가 건수 포함
전제 모든 유입에 즉시 응대 가능 비현실적 가정
결과 항상 50~90% 성과 측정 불가능

본 분석을 해석할 때 다음 사항을 유의해야 합니다.
항목 한계/유의사항 영향
근무시간 정의 원본 데이터는 09:00~18:00 기준이나, 시간대 구간이 7~9시/10~12시로 구분되어 있어 09시가 어느 구간에 포함되는지 불명확. 본 분석에서는 10~17시를 근무시간으로 보수적 계산함. 실제 근무시간 내 유입이 약간 더 높을 수 있음
정상 부재율 기준 방문 20~30%, 비대면 40~50%는 사전 합의된 기준값으로 제공됨. 실제 계산 결과가 아닌 목표/판정 기준임. 실측 데이터와 차이가 있을 수 있음
근무외 유입 ≠ 100% 부재 근무시간 외 유입도 익일 콜백을 통해 연결 가능. 다만 시간 경과에 따른 고객 관심 저하로 연결률이 낮아지는 경향 존재. 구조적 부재율 추정치보다 실제 부재율이 낮을 수 있음
방문 상담 부재의 정의 방문 상담에서 "부재"가 무엇을 의미하는지 명확하지 않음 (예약 후 미방문? 상담 미완료?). 비대면과 동일 기준 적용 가능 여부 검토 필요. 방문/비대면 부재율 직접 비교 시 주의
결론적 유의사항: 본 분석은 유입 시간대 구조가 부재율에 미치는 영향을 설명하는 구조적 진단이며, 상담팀 성과가 양호하다는 것을 보장하지는 않습니다. 정상 부재율 기준 도입 후에도 근무시간 내 응대 품질에 대한 별도 검증이 필요합니다.

핵심 인사이트 5줄 요약

  1. 비대면 상담 유입의 53.7%는 근무시간 외(18~09시)에 발생하며, 이 시간대에는 즉시 응대가 물리적으로 불가능함.
  2. 상담카드(유입량)와 응대율은 역상관: 유입량 2.4배 증가 시, 응대율은 절반으로 감소(19.4%→9.5%). 응대된 건의 전환율은 68%로 양호함.
  3. 마케팅 효율성 역설: 60% 더 많은 유입(1~3월 3,919건→11~12월 6,281건)이 54% 적은 구매(566건→263건)로 이어짐.
  4. 적정 유입량 존재: 월 1,200~1,600건 수준에서 응대율(15~22%)과 전환율(70~85%)이 최적화됨. 초과 시 90%가 미응대.
  5. 부서간 협업 필요: 마케팅팀은 유입량이 아닌 구매 전환 기준 KPI 설정, 상담팀은 정상 부재율 기준 KPI 도입 권고.

한 문장 결론 (Decision Sentence)

"유입량 60% 증가가 구매 54% 감소로 이어지는 구조에서, 마케팅팀은 적정 유입량(월 1,200~1,600건) 관리를, 상담팀은 '정상 부재율' 기반 KPI를 도입하여, 양 부서가 '구매 전환'이라는 공동 목표 하에 협업해야 한다."
