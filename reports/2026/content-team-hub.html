01 개요

콘텐츠 허브는 팀의 콘텐츠 일정을 효율적으로 관리하고 협업할 수 있는 올인원 솔루션입니다.

캘린더 뷰

월별 캘린더에서 콘텐츠 일정을 한눈에 확인

게시판 뷰

리스트 형태로 콘텐츠를 정렬 및 필터링

콘텐츠 관리

등록, 수정, 복제, 삭제 및 휴지통 복원

메모 협업

일/주/월별 메모로 팀원과 실시간 소통

디바이스 미리보기

데스크탑, 태블릿, 모바일 화면 미리보기

다크/라이트 테마

눈의 피로를 줄이는 테마 전환 지원

02 캘린더 뷰

월별 캘린더에서 콘텐츠 일정을 시각적으로 확인할 수 있습니다.

1

월 이동

상단의 ◀ ▶ 버튼으로 월을 이동합니다. 월/주/일 버튼으로 보기 모드를 변경할 수 있습니다.

2

콘텐츠 클릭

캘린더에 표시된 콘텐츠를 클릭하면 상세 정보 모달이 열립니다. 수정, 삭제, 복제가 가능합니다.

3

유형별 색상 구분

콘텐츠 유형에 따라 다른 색상으로 표시됩니다.

인스타그램 유튜브 블로그 홈페이지 기타
4

지난 콘텐츠 표시

오늘 이전 날짜의 콘텐츠는 그레이 색상으로 자동 변환됩니다.

03 게시판 뷰

리스트 형태로 콘텐츠를 확인하고 정렬할 수 있습니다.

1

게시판 뷰 이동

사이드바에서 게시판 뷰 메뉴를 클릭합니다. (단축키: 2)

2

기간 필터

월/주/일 버튼으로 표시할 기간을 선택합니다.

3

정렬 기능

정렬 버튼을 클릭하여 다양한 기준으로 정렬할 수 있습니다.

  • 최신순: 최근 날짜부터 표시
  • 오래된순: 과거 날짜부터 표시
  • 유형순: 콘텐츠 유형별 그룹화
  • 상태순: 예정 → 진행중 → 완료 순서
4

콘텐츠 상세

테이블 행을 클릭하면 콘텐츠 상세 모달이 열립니다.

04 콘텐츠 관리

새 콘텐츠 등록

1

등록 버튼 클릭

새 콘텐츠 등록 버튼을 클릭하거나 단축키 N을 누릅니다.

2

필수 정보 입력

* 표시된 필드는 필수 입력 항목입니다.

  • 콘텐츠 유형: 인스타그램, 유튜브, 블로그 등
  • 제목 *: 콘텐츠 제목 (필수)
  • 설명: 간단한 설명 또는 이슈사항
  • 링크: 콘텐츠 URL
  • 담당자: 팀원 중 담당자 지정
  • 발행 예정일 *: 날짜 선택 (필수)

콘텐츠 상세 화면

캘린더나 게시판에서 콘텐츠를 클릭하면 상세 화면이 열립니다.

수정

날짜, 상태, 수정자, 설명, 링크를 변경할 수 있습니다.

복제

콘텐츠를 복사하여 새로운 일정으로 등록합니다.

휴지통

삭제된 콘텐츠는 휴지통으로 이동되며 복원이 가능합니다.

  • 복원: 휴지통에서 원래 위치로 복원
  • 영구 삭제: 완전히 삭제 (복구 불가)
  • 휴지통 비우기: 모든 항목 영구 삭제
Tip! 필수 항목을 입력하지 않으면 빨간 테두리와 함께 안내 메시지가 표시됩니다.

05 메모 기능

메모 기능으로 팀원들과 실시간으로 협업하고 중요 사항을 기록할 수 있습니다.

1

메모 탭 이동

우측 패널에서 메모 탭을 클릭합니다.

2

메모 작성

하단 입력창에 메모 또는 이슈 사항을 입력하고 등록 버튼을 클릭합니다.

3

기간별 필터링

캘린더의 특정 날짜나 월을 선택하면 해당 기간의 메모만 표시됩니다.

  • 일별: 특정 날짜의 메모만 보기
  • 주별: 선택한 주의 메모 보기
  • 월별: 현재 표시된 월의 메모 보기
4

메모 다운로드

메모 영역 하단의 다운로드 버튼으로 기간별 메모를 텍스트 파일로 저장할 수 있습니다.

메모 넘버링

메모가 7개 이상이 되면 자동으로 번호가 표시되어 관리가 편리해집니다.

06 미리보기

발행 전 다양한 디바이스에서 콘텐츠가 어떻게 보이는지 미리 확인할 수 있습니다.

1

미리보기 탭 선택

우측 패널에서 미리보기 탭을 클릭합니다.

2

URL 입력

확인하고 싶은 콘텐츠의 URL을 입력합니다.

3

디바이스 선택

데스크탑, 태블릿, 모바일 중 원하는 화면 크기를 선택합니다.

데스크탑
태블릿
모바일
4

미리보기 실행

미리보기 버튼을 클릭하면 선택한 디바이스 크기로 콘텐츠가 표시됩니다.

Tip! 일부 외부 사이트는 보안 정책으로 미리보기가 제한될 수 있습니다. 이 경우 "검색하기" 버튼을 이용해 새 창에서 확인하세요.

07 테마 및 단축키

다크/라이트 테마

눈의 피로를 줄이고 작업 환경에 맞는 테마를 선택할 수 있습니다.

  • 헤더의 / 아이콘을 클릭하거나 단축키 T를 누릅니다.
  • 선택한 테마는 브라우저에 저장되어 다음 방문 시에도 유지됩니다.

키보드 단축키

자주 사용하는 기능을 빠르게 실행할 수 있습니다. ?를 눌러 단축키 목록을 확인하세요.

08 데이터 관리

모든 콘텐츠와 메모 데이터는 브라우저에 자동 저장되며, 로딩 중에는 스피너가 표시됩니다.

자동 저장

  • 콘텐츠 등록/수정/삭제
  • 메모 등록/삭제
  • 휴지통 이동/복원

저장 용량

  • 약 6개월분 데이터 저장
  • 오래된 데이터 자동 정리

백업 및 복원

사이드바 하단의 버튼으로 데이터를 관리할 수 있습니다.

  • 백업: JSON 파일로 전체 데이터 다운로드
  • 복원: 백업 파일로 데이터 복원
Tip! 주 1회 정기 백업을 권장합니다. 브라우저 캐시 삭제 시 데이터가 삭제될 수 있습니다.

주의사항

데이터는 현재 브라우저에만 저장됩니다. 다른 기기나 브라우저에서는 공유되지 않습니다.
