# Clipdesk 브랜드 통합 전략 보고서

timbel의 Clipdesk One Brand 전략 수립 및 홈페이지 IA 설계를 위한 전략 보고서 웹 문서입니다.

## 프로젝트 개요

- **작성 목적**: 브랜드 통합 전략 분석 및 제안
- **요청**: 경영진
- **작성자**: 이세영 팀장
- **문서 버전**: v1.1 (KOBA 2026 추가)

## 파일 구조

```
_website/
├── clipdesk-strategy-report.html   # 메인 전략 보고서
├── README.md                        # 프로젝트 설명
└── STRATEGY-STRUCTURE.md           # IA 구조 문서
```

## 실행 방법

1. `clipdesk-strategy-report.html` 파일을 브라우저에서 열기
2. 좌측 네비게이션 메뉴를 통해 각 섹션 탐색

## 주요 기능

### 반응형 디자인
- 데스크톱: 좌측 고정 메뉴 + 우측 콘텐츠 뷰포트
- 모바일: 햄버거 메뉴 + 슬라이드인 네비게이션

### 네비게이션
- 섹션 앵커 링크 (smooth scroll)
- 현재 섹션 하이라이트 (IntersectionObserver)
- 프로젝트 단계 뱃지 표시

### 디자인 시스템
- Toss 스타일 기반 미니멀 UI
- Pretendard 폰트
- 블루 포인트 컬러 (#3182F6)
