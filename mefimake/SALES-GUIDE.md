## 🛡️ 설득 전략

### 1. 리스크 분산 구조 (핵심)

```
기존 인식: 300만원 = 전부 잃거나 / 전부 성공
전환 인식: 100만원 × 3단계, 각 단계마다 가치 획득

┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
│ 1단계    │→ │ 2단계    │→ │ 3단계    │→ │ 4단계    │
│ 적성확인  │  │ 기초학습  │  │ 자격취득  │  │ 수입발생  │
│ (무료)   │  │ (100만)  │  │ (+100만) │  │ (+100만) │
└──────────┘  └──────────┘  └──────────┘  └──────────┘
     ↓              ↓              ↓              ↓
"안 맞으면      "기초만으로도    "자격증은       "여기까지 오면
 여기서 Stop"   평생 스킬"      평생 유효"      성공"
```

**💬 멘트:**
> "300만원 전체를 걸지 마세요. 
> 단계별로 확인하시면 됩니다.
> 1단계 적성 확인은 무료예요.
> 안 맞으면 거기서 멈추시면 됩니다."

---

### 2. 안전장치 명시

| 장치 | 설명 | 멘트 |
|------|------|------|
| 환불 정책 | 1주일 내 100% | "1주일 들어보시고 아니다 싶으면 전액 돌려드려요" |
| 분할 납부 | 월 25만원 × 12 | "하루 8천원, 커피 2잔 값이에요" |
| 중도 전환 | 다른 과정 가능 | "안 맞으면 다른 과정으로 전환할 수 있어요" |
| 평생 복습 | 기간 제한 없음 | "한 번 결제하면 평생 접근 가능해요" |
| 취업 연계 | 못 하면 추가 지원 | "취업까지 무상 연장 지원해드려요" |

### 3. 비교 기준 재설정

#### A. 시간 비용

| 선택지 | 기간 | 시간 가치 환산 |
|--------|------|--------------|
| 혼자 독학 | 12~24개월 | 960만원 (주 10시간 × 시급 2만원) |
| 체계적 학습 | 3~4개월 | 300만원 |
| **절약** | 8~20개월 | **640만원 시간 가치** |

**💬 멘트:**
> "무료로 혼자 하면 시간이 960만원어치 들어가요.
> 300만원 투자하면 640만원 시간을 아끼는 거예요."

#### B. 기회비용

**💬 멘트:**
> "1년 전에 시작했다면 지금 어디 있을까요?
> 6개월 후에 '그때 시작할걸' 하실 수도 있어요.
> 300만원은 다시 모을 수 있지만,
> 시도하지 않은 시간은 돌아오지 않습니다."

#### C. 재도전 가능성

**💬 멘트:**
> "이건 되돌리기 가능한 결정이에요.
> 1주일 내면 환불, 중간에 전환도 가능해요.
> 이사나 이직 같은 큰 결정이 아니에요."

---

### 4. 실패 재정의

```
❌ 기존 정의: 수입 발생 안 하면 = 실패
✅ 새 정의: 안 맞는다는 걸 알았으면 = 값진 발견

❌ 기존 정의: 중간에 포기하면 = 실패
✅ 새 정의: 시도해본 경험 = 다음 선택에 활용
```

**💬 멘트:**
> "300만원으로 '안 맞는 길'을 확인했다면,
> 앞으로 10년간 방황할 시간을 아낀 겁니다."

---

### 5. 진입 장벽 낮추기

#### 필요하지 않은 것

| 걱정 | 현실 |
|------|------|
| "타이핑 느려요" | 200타 이하 시작 47% |
| "전공 아니에요" | 전공자 비율 12%뿐 |
| "나이가 많아요" | 30~40대가 62% |
| "시간이 없어요" | 육아 중 시작 34% |

#### 필요한 것 (체크리스트)

- ✅ 하루 1~2시간 투자 의지
- ✅ 기본 컴퓨터 사용 가능
- ✅ 3개월간 꾸준히 학습할 환경
- ✅ 새로운 것을 배우려는 마음

**💬 멘트:**
> "완벽하지 않아도 됩니다.
> 타이핑 200타 이하에서 시작한 분이 47%예요.
> 최고령 수강생은 58세, 지금 프리랜서 2년차입니다."

---

### 6. 성공의 재정의

```
🏁 레벨 1: 시작했다 = 성공
   → "시작한 것 자체가 상위 10%"

🏁 레벨 2: 1개월 버텼다 = 성공
   → "여기까지 온 것만으로도 대단해요"

🏁 레벨 3: 자격증 취득 = 성공
   → "평생 유효한 자산이 생겼습니다"

🏁 레벨 4: 첫 수입 발생 = 성공
   → "10만원이든 100만원이든, 첫 수입은 첫 수입"

🏁 레벨 5: 월 100만원+ = 성공
   → "본격적인 부업 레벨"

🏁 레벨 6: 월 300만원+ = 큰 성공
   → "본업 전환 가능 레벨"
```

**💬 멘트:**
> "월 300만원이 목표가 아니어도 됩니다.
> 먼저 레벨 1부터 시작하세요.
> 시작하는 것 자체가 상위 10%입니다."

## 📞 상담 핵심 질문 5가지

### 질문 1: 동기 확인

> **"왜 속기사에 관심을 갖게 되셨어요?"**

**목적:** 진짜 동기 vs 순간적 관심 구분

| 좋은 답변 | 주의 필요 |
|----------|----------|
| "재택으로 일하고 싶어서" | "그냥 좋아 보여서" |
| "부수입이 필요해서" | "광고에서 봐서" |
| "경력 전환을 고민 중" | "뭐라도 해야 할 것 같아서" |

**후속 질문:**
> "그 상황이 얼마나 급하신가요?"
> "이 문제를 해결하지 않으면 6개월 후엔 어떨 것 같으세요?"

---

### 질문 2: 과거 경험 확인

> **"이전에 비슷한 자기계발이나 강의를 들어보신 적 있으세요?"**

**목적:** 실패 트라우마 파악 + 공감

**대응:**
```
고객: "네, 몇 번 들었는데 다 실패했어요"

상담사: "그렇군요. 그때 왜 안 됐다고 생각하세요?"
       (→ 경청)
       "그 경험이 있으니까 이번엔 뭐가 달라야 할지 아시겠네요."
       "저희는 그 부분을 이렇게 해결합니다..."
```

---

### 질문 3: 현실 조건 확인

> **"하루에 어느 정도 시간을 투자할 수 있으세요?"**

**목적:** 현실적 가능성 체크 (무리한 등록 방지)

| 답변 | 대응 |
|------|------|
| "1~2시간" | ✅ "충분합니다" |
| "30분 미만" | ⚠️ 솔직하게 안내 |

**솔직한 안내 (30분 미만일 때):**
> "솔직히 하루 30분으로는 어렵습니다.
> 1시간 이상 확보 가능하실 때 다시 연락 주세요.
> 그때도 저희는 여기 있을 거예요."

**💡 이 솔직함이 역설적으로 신뢰를 높입니다.**

---

### 질문 4: 두려움 직접 질문

> **"솔직히 가장 걱정되는 부분이 뭐예요?"**

**목적:** 숨겨진 반대 의견(objection) 표면화

| 고객 반응 | 대응 |
|----------|------|
| "돈이 아까울까봐요" | "맞아요. 그래서 1주일 환불 정책이 있어요. 1주일 들어보시고 '이건 아니다' 싶으면 전액 돌려드려요." |
| "내가 끝까지 할 수 있을까요?" | "73%가 완주합니다. 혼자 하면 10%도 안 돼요. 왜냐하면 저희가 계속 챙기거든요." |
| "가족이 반대할 것 같아요" | "가족분이 걱정하시는 게 뭘까요? 필요하시면 가족분과 함께 통화해도 괜찮아요." |

---

### 질문 5: 결정 시점 확인

> **"오늘 바로 결정하실 필요는 없어요. 
> 충분히 생각해보시고, 언제쯤 결정하실 것 같으세요?"**

**목적:** 압박 없이 타임라인 설정

**효과:**
- ✅ 압박감 제거 → 신뢰 상승
- ✅ 구체적 시점 확보 → 팔로업 가능
- ✅ "나중에"가 아닌 "언제"로 전환

**후속:**
> "좋아요. 그때까지 더 궁금한 거 있으시면 편하게 카톡 주세요.
> 제가 OO일에 한 번 더 연락드려도 될까요?"
