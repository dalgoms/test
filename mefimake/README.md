## 🔄 반복 가능한 프로세스

### 월간 광고 소재 생성 워크플로우

```
STEP 1: 캠페인 설정
    ↓
    주제/직업 입력 (예: 속기사)
    타겟 선택 (취준생, 직장인, 주부)
    ↓
STEP 2: 콘텐츠 매트릭스 확인
    ↓
    content-matrix.js에서 해당 주제 콘텐츠 확인
    없으면 자동 생성 or 새로 추가
    ↓
STEP 3: 템플릿 × 테마 조합
    ↓
    6개 템플릿 × 6개 테마 = 36개 조합 가능
    자동 로테이션으로 다양성 확보
    ↓
STEP 4: 배치 생성
    ↓
    20개 이미지 자동 생성
    주차별 분배 (주당 5개)
    ↓
STEP 5: 다운로드 & 업로드
    ↓
    PNG 파일 다운로드
    Meta 광고 관리자에 업로드
```

## 📊 월간 콘텐츠 계획 예시 (20개)

### 주차별 배분

| 주차 | 템플릿 분배 | 목적 |
|------|------------|------|
| 1주차 | 혜택 2, 질문 2, 스토리 1 | 인지도 확보 |
| 2주차 | 통계 2, 비교 2, 혜택 1 | 신뢰 구축 |
| 3주차 | 스토리 2, 질문 2, 긴급 1 | 감정 연결 |
| 4주차 | 긴급 2, 혜택 2, 통계 1 | 전환 유도 |

### 비율 로테이션

- 1:1 (피드): 40%
- 4:5 (피드 권장): 40%
- 9:16 (스토리/릴스): 20%

## 🚀 사용 방법

### 1. 빠른 시작

1. `index.html`을 브라우저에서 열기
2. 주제 입력 (예: 속기사)
3. "전체 배치 생성" 클릭
4. 20개 이미지 자동 생성 및 다운로드

### 2. 커스터마이징

1. 원하는 템플릿 선택
2. 컬러 테마 선택
3. 훅/메인/서브 카피 수정
4. "현재 이미지 다운로드" 클릭

### 3. 콘텐츠 라이브러리 활용

1. 좌측 패널 하단의 "콘텐츠 라이브러리" 열기
2. 훅 카피 / 혜택 / CTA 탭 전환
3. 원하는 항목 클릭하여 적용

## 💡 팁

### 효과적인 A/B 테스트

1. 같은 콘텐츠로 다른 테마 테스트
2. 같은 테마로 다른 템플릿 테스트
3. 같은 템플릿으로 다른 훅 카피 테스트

### 타겟별 최적화

- **취준생**: 미래 지향적 메시지, 통계형 선호
- **직장인**: 부수입 강조, 비교형 효과적
- **주부**: 재택/유연성 강조, 스토리텔링 효과적

## 🔧 확장 가능성
